Survival Timer How to make a survival timer for your Lens Studio games

Survival timer

Hello guys, in this video, I'm talking about how I made this survival timer for my zombie-killing game.

When we initiate the game by tapping the start button, the survival timer begins. If the zombies get to us or succeed in killing us, the survival timer stops. This linkage is established through the game over image object scene. In simpler terms, when the game over image object is enabled, , the survival timer also comes to a halt. This functionality is implemented using a sample JavaScript code which I wrote along with the necessary Lens Studio behavior script.

Watch the video you will understand more fore this survival timer, I've just created a sample prefab. which i used in this video tutorial . I hope it proves to be useful. Additionally, You can tap download button to download it. Feel free to download and use it for your projects or modify it as needed.
